# Proyecto de Bot de Búsqueda y Extracción de Información

Este proyecto consiste en un bot de consola desarrollado en Node.js que permite realizar diversas operaciones de búsqueda y extracción de información desde la línea de comandos. Fue creado como parte de un proyecto universitario para demostrar habilidades en desarrollo de software utilizando tecnologías como Node.js, Axios y Cheerio.

## Funcionalidades

### 1. Búsqueda en Wikipedia

Permite realizar búsquedas en Wikipedia y muestra información relevante, incluyendo títulos, descripciones y URLs de los resultados.

### 2. Último Encabezado de Noticias de Televisa.com

Extrae el último encabezado de noticias de la página www.televisa.com utilizando técnicas de web scraping.

### 3. Obtención de Precio desde una Página de Compras

Permite ingresar la URL de una página de compras y extrae el precio de manera básica (requiere adaptación según la estructura específica de la página).

## Instrucciones de Uso

1. Clona este repositorio a tu máquina local.
2. Instala las dependencias ejecutando `npm install`.
3. Ejecuta el programa con `npm run start` y sigue las instrucciones en la consola para utilizar las funciones.

## Requisitos

- Node.js instalado en tu máquina.

## Notas Importantes

- Este proyecto fue creado con fines educativos y para demostrar habilidades en desarrollo de software.
- Ten en cuenta las consideraciones éticas y legales al utilizar técnicas de web scraping, y asegúrate de cumplir con los términos de servicio de los sitios web consultados.
